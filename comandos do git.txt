Comandos do git: 
git config --global user.name "Fulano de Tal" - loga o username
git config --global user.email fulanodetal@exemplo.br - loga o email
git init - inicia um repositório na sua máquina
git clone - pega o repositório do servidor e clona na máquina
ls - lista os arquivos dentro da pasta
git pull origin - pega os arquivos do repositório no servidor e passa para a máquina
cd - change directory
cd .. - volta um nível 
git status - mostra o status do arquivo (staged, commit..)
git add . - prepara as mudanças para serem enviadas ao servidor
git commit -m - envia as mudanças para o servidor ("-m" adiciona uma mensagem)
git push origin main/master - copia o arquivo da máquina para o repositório.   
git remote add origin url do repositório no github
